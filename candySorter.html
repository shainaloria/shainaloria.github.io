Candy
Color Sorter

An Arduino project built to arrange candies (specifically skittles) according to their corresponding colors. This was constructed using a color sensor (TCS3200), two servo motors (SG90) and the Arduino Uno. The body was created using an illustration board and a plastic straw as a tube for the Skittles to go through.
